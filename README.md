# Ignews

Esse repositório consiste no projeto de site de notícias Ignews, desenvolvido durante o treinamento Ignite da Rocketseat.

# Recursos utilizados

- Typescript
- NextJS
- ReactJS
- Sass
- Axios
- Prismic
- Stripe
- FaunaDB
